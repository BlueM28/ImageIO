# ImageIO

画像の入出力を勉強するために作成したソースコードです。
皆様の学習の役に立っていただければと思います。

Makefileも同梱しており、ビルドすることは可能ですが、実用性のあるプログラムではありません。
ソースコードを閲覧していただくことが目的です。

ここのコードを使って解説もしています。
